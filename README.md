# Aota 2

**Aota 2 - игра по мотивам всеми известной Dota 2. Однако в этой игре предстоит не защищать крепость, а наоборот атаковать её.
В игре будет присутствовать возможность прокачки героя и покупки оружия, сражение с NPC. Игра будет происходить на большой карте.**

## ТЗ
* Создание БД:
    * Реализовать таблицу для никнеймов игроков и их результатов.
    * Вносить первый и/или лучший результат игроков.
    * Написать функции, протестировать.
* Общая идея:
    * Продумать суть игры, стиль, сюжет.
    * Найти все необходимые спрайты и текстуры.
* Визуальная составляющая:
    * Создать главное окно, окно лидерборда, финальное окно с отображением результатов.
    * Добавить музыкальное сопровождение.
* Сама игра:
    * Дать игрокам возможность придумать свой ник.
    * Создать основной файл игры.
    * Создать отдельные файлы для определённых игровых классов.
    * Создать папку data для хранения в ней спрайтов, текстур и картинок.
    * Добавить игрока, дать ему возможность передвигаться по карте и проводить все необходимые действия с другими игровыми объектами.
    * Добавить врагов, которые будут двигаться в направлении игрока и атаковать его.
    * Добавить главное здание, которое надо будет уничтожить.
* Соединить все компоненты(файлы) и протестировать.
